# Pricing Intelligence System Design

## Overview

A system that uses Claude API to scrape vendor websites, extract pricing data, and generate comprehensive pricing guides with market trends and competitive analysis.

## Data Sources

### 1. Vendor Websites
- Cisco Commerce Workspace
- Microsoft Volume Licensing
- VMware Store
- Poly/Plantronics pricing pages
- Logitech B2B portal
- Yealink partner portal

### 2. Distributor APIs
- Ingram Micro (already integrated)
- Tech Data / TD Synnex
- D&H Distributing

### 3. Market Intelligence
- Industry reports (Gartner, IDC)
- Public price lists
- Competitor quotes (anonymized)
- Historical trends

## Claude API Integration

### Use Cases

#### 1. Price Extraction
**Prompt**:
```
Analyze this vendor pricing page and extract all product prices.

HTML Content: [page content]

Extract and return in JSON format:
{
  "products": [
    {
      "name": "product name",
      "sku": "part number",
      "msrp": 999.99,
      "list_price": 899.99,
      "currency": "USD",
      "availability": "in stock",
      "notes": "any special pricing info"
    }
  ]
}
```

#### 2. Price Trend Analysis
**Prompt**:
```
Analyze these price snapshots and generate market insights:

Product: Cisco Catalyst 9300 Switch
Historical Prices:
- Jan 2026: $8,500
- Feb 2026: $8,200
- Mar 2026: $7,950

Provide:
1. Trend direction and magnitude
2. Likely causes (market conditions, product lifecycle, competition)
3. Future price prediction
4. Buying recommendation
```

#### 3. Competitive Analysis
**Prompt**:
```
Compare pricing across vendors for similar products:

Product Category: Unified Communications Platform
- Cisco Webex: $15/user/month
- Microsoft Teams: $12.50/user/month
- Zoom Phone: $10/user/month
- RingCentral: $19.99/user/month

Analyze:
1. Market positioning
2. Value proposition differences
3. TCO considerations
4. Recommendation by use case
```

#### 4. PDF/Document Analysis
**Prompt**:
```
Extract pricing information from this vendor price list PDF.

[PDF content or image]

Return structured data with:
- Product categories
- SKUs and descriptions
- List prices and discount tiers
- Special promotions
- Effective dates
```

- Rate limit requests

## Example Workflows

### Workflow 1: Daily Price Update
1. Cron job triggers at 2 AM daily
2. Fetch list of products needing updates
3. For each product:
   - Scrape vendor website
   - Extract HTML + take screenshot
   - Send to Claude for analysis
   - Parse and validate results
   - Store in price_snapshots
   - Update pricing_guides with new average
4. Calculate trends for updated products
5. Generate insights for significant changes
6. Send email report to admins

### Workflow 2: Manual Price Research
1. Admin enters product name in search
2. System finds relevant vendor pages
3. Initiates scrape jobs for each source
4. Claude analyzes all sources
5. System aggregates results
6. Displays comparison table with:
   - Source URLs
   - Extracted prices
   - Confidence scores
   - Recommendations
7. Admin approves and saves to pricing guide

### Workflow 3: Competitive Analysis Report
1. Admin selects product category
2. System pulls all products in category
3. Groups by product type/tier
4. Sends to Claude for competitive analysis
5. Claude generates:
   - Market positioning chart
   - Price comparison matrix
   - TCO analysis
   - Buying recommendations
6. System generates PDF report
7. Email to sales team

## Cost Estimation

### Claude API Costs
**Assumptions**:
- 1000 products tracked
- Daily updates
- ~2000 tokens per analysis

**Monthly costs**:
- Input tokens: 1000 products × 2000 tokens × 30 days = 60M tokens
- Output tokens: ~15M tokens
- **Estimated**: ~$200-300/month for comprehensive analysis

**Optimization**:
- Only update changed products
- Use caching for similar products
- Batch requests
- Use Claude Haiku for simple extractions

### Infrastructure Costs
- Screenshot service: Free (Puppeteer/Playwright)
- Storage: ~$5/month (S3 for screenshots)
- Compute: Included in existing hosting

**Total**: ~$205-305/month for full pricing intelligence
